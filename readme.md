# 自动驾驶系统的可视化

qt作为交互界面，opengl作为可视化API.

# 依赖
* ubuntu 16.04
* protobuf 3.18.0
* qt 5.5.1 (sudo apt-get install qt5-default)
* eigen
* google test 1.11.0
* glog 0.50
* benchmark 1.6.0
* openscenegraph 3.2.1
